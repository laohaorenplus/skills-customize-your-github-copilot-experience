
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. You'll practice string manipulation, loops, conditionals, and random selection while creating an interactive game.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. The game should randomly select words, process user guesses, track progress, and determine win/lose conditions.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress (e.g., `_ _ _` format)
- Track incorrect guesses remaining
- End when the word is guessed or attempts are exhausted
- Display win/lose messages to the player
