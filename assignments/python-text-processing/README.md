# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master Python string manipulation, file I/O operations, and text processing techniques. You'll build practical tools for processing and transforming text data, including reading from files, manipulating strings, and writing results back to files.

## 📝 Tasks

### 🛠️ Text Processing with Strings

#### Description
Create a program that performs various string operations and text transformations. Build functions to work with strings, including searching, replacing, formatting, and analyzing text data.

#### Requirements
Completed program should:

- Implement string searching and pattern matching
- Create functions to replace and transform text
- Handle uppercase, lowercase, and case conversion
- Count occurrences of words or characters in a string
- Split and join strings appropriately
- Work with string slicing and indexing

### 🛠️ File I/O and Text Manipulation

#### Description
Build a program that reads text from a file, processes it according to specific rules, and writes the results to a new file. Your program should handle real-world text processing tasks like filtering, sorting, or formatting data.

#### Requirements
Completed program should:

- Read data from an input text file
- Process the file contents using string methods
- Filter or transform lines based on specific criteria
- Write processed results to an output file
- Handle file paths correctly
- Include error handling for file operations
